# Import Markdown

Rise4fun automatically add support for importing patial markdown files using [remark-import-partial](https://github.com/dotansimha/remark-import-partial).

```markdown
Some content.

{@import ./partial-demo._md}

More content.
```

The file path is relative to the current file location.

<BrowserWindow>

Some content.

{@import ./partial-demo._md}

More content.

</BrowserWindow>
