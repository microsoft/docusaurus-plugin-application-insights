# Theme Side Editor

A Docusaurus theme component that loads a code editor in a split pane.

## Configuration

Install the theme

```bash npm2yarn
npm install @rise4fun/docusaurus-theme-side-editor
```

Add the theme to your theme list.

```js
const config = {
    plugins: [
        // highlight-start
        ["@rise4fun/docusaurus-theme-side-editor", {

        }]
        // highlight-end
    ]
}
```
