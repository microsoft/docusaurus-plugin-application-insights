# Introduction

This projectn provides tooling to create documentation of programming tools created
by the Microsoft RiSE team.

The project contains various plugins and customization of the [Docusaurus](https://docusaurus.io/)
web site.

The goal of the project is to allow researcher to publish high quality, modern documentation for their tools.

    - use best-in-class documentation framework Docusaurus
    - execution of tools at build time
    - compliance with privacy, accessibility out of the box

## Rise4fun history

Historically, the RiSE published tools through a centralized web site, `https://rise4fun.com/`.
Rise4fun was eventually retired and this effort was put in place to revive the effort.
